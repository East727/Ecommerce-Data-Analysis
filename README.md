# Ecommerce-Data-Analysis
# 跨境电商订单数据分析（个人项目）

## 业务背景
分析英国电商2010-2011年销售数据，探索销售趋势、客户行为、产品表现，为运营提供建议。

## 工具与技能
- Python（pandas）：数据清洗、缺失值处理、特征工程
- SQL：多表聚合、分组统计、客户分层
- PowerBI：交互式仪表盘、DAX度量值

## 核心结论
- 总销售额xxx万英镑，11月为销售高峰
- 英国占80%销售额，荷兰、爱尔兰客单价最高
- 畅销品类：礼品、装饰品
- 回头客占比xx%

## 文件说明
- data_cleaning.py：数据清洗脚本
- queries.sql：SQL查询语句
- dashboard.pbix：PowerBI可视化文件
- report.pdf：分析报告

## 如何运行
1. 下载Online Retail数据集
2. 运行data_cleaning.py生成清洗后csv
3. 用PowerBI打开dashboard.pbix
